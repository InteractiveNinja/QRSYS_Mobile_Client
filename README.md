## React QRSYS Mobile Client

Applikation die beim sich beim Server nach Verbunden Devices erkundigt. Mit den Angaben der BenutzerID des Eingeloggten Benutzers können diese Devices abgefragt werden. Anhand der Device ID und der Benutzer ID kann ein Gerät eine Nachricht geschickt werden.

### Features

* Abfrage von Geräten anhand seiner Benutzer ID

* Versenden von Nachrichten anhand der Benutzer ID und Device ID
